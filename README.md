# StreamRed 🎥

A self-hosted **Netflix-style streaming platform** that plays videos directly from Google Drive. Built with Flask, SQLite, and Google Drive API.

![StreamRed Banner](https://images.unsplash.com/photo-1594909122845-11baa439b7bf?w=800)

## ✨ Features

- 🎬 Beautiful Netflix-like UI
- 🔄 Real-time library refresh from Google Drive
- 📱 Responsive design (Mobile + Desktop)
- 👤 Simple user authentication
- 🔀 **Multiple Player Options** (Preview, Direct, HTML5)
- 📊 Continue Watching + Most Viewed sections
- ⚡ Fast & lightweight

- **Database**: SQLite
- **Storage**: Google Drive
- **Frontend**: HTML, CSS, JavaScript
- **Deployment**: GitHub Codespaces / Any VPS
